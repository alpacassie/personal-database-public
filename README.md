# Personal Database Public

A public-facing version of Cassie's personal database format, focused on shareable skills, prompts, and small AI workflows.

Live site: coming soon

## What is inside

- Skills database
- Prompt database with copy-ready prompts
- Workflow database for small repeatable systems

## Run locally

```bash
python3 -m http.server 5002
```

Then open <http://127.0.0.1:5002/#skills>.
