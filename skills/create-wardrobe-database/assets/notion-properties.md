# Notion Wardrobe Database

Create a database with these properties:

- Name: title
- Product name: text
- Brand: text
- Type: select with top, bottom, outerwear, dress, shoes
- Subtype: text or select
- Color: text or select
- Season: select with summer, winter, all-season
- Capsule: checkbox
- Notes: text

Suggested views:

- All pieces
- Capsule only
- By type
- By season
